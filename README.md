# Political System Game

Ein spielbarer React/Vite-Prototyp fuer eine politische Strategie- und Regierungssimulation.

## Lokal starten

```bash
npm install
npm run dev
```

## Produktionsversion bauen

```bash
npm run build
```

## Online deployen

Empfohlen: Vercel oder Netlify.

Build Command:

```bash
npm run build
```

Output Directory:

```bash
dist
```
